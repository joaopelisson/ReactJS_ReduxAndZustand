## Getting Started

### Prerequisites

- NodeJS
- Yarn or NPM

### Installation

1. Clone the repository: 

```
git clone https://github.com/joaopelisson/ReactJS_ReduxAndZustand
```


2. Install the necessary packages:
```
npm i
or
yarn install
```

3. Run the project:
````
npm run dev
//and
npm run server 
````
